HTML tables

Table of contents:



Element

Explanation
table

Table is the element that initializes and construct a table in HTML.
td

td stands for table data. It is essentially the table data that goes into a column.
tr

tr stands for table row. It is what defines a row in your table. td elements go inside of it.
th

th stands for table head. It is used as the head of the table, showing what that data in the row or column is about.

thead

It defines a table head for a table. Usually, tr and th elements go inside it.
tbody

It defines a table body for the table. Usually, tr and td elements go inside it. This is where the main big chunck of data will go into your table.
rowspan

The rowspan defines how many rows (vertically) your table box will encapsulate. Its default set is 1.
colspan

The colspan defines how many rows (horizontally) your table box will encapsulate. Its default set is 1.
tfoot

The tfoot defines a footer for your table to summarize the data content in the column.
